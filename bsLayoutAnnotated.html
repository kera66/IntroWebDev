Image Alignment Examples

Tutorials  --> Getbootstrap --> W3Schools (scroll down to Aligning Images)

For images we now always use the .img-fluid class with all our <img> tags.
For example, <img class="img-fluid">

And we no longer use height="xx" or width="xx"

To center an image we use the classes .mx-auto .d-block
For example, <img class="img-fluid mx-auto d-block">

To left align an image use the class .float-left For example, <img class="float-left">
To right align an image use the class .float-right For example, <img class="float-right">

Note, these classes work for images but do not include padding or margins.


class="float-left"


class="mx-auto d-block"


class="float-right"



Responsive Embedded Video

Tutorials  --> Getbootstrap --> W3Schools (scroll down to Responsive Embeds)

To embed a video from YouTube or Vimeo (Step 1) use the class .embed-responsive-item in the <iframe> tag.
For example: <iframe class="embed-responsive-item">

And then (Step 2) remove the height="xx" and width="xx from all your YouTube or Vimeo <iframe>'s.

And then (Step 3) wrap the <iframe class="embed-responsive-item"> tag from step 1 in another <div> tag.
For example: <div class="embed-responsive embed-responsive-16by9">

NavBar Examples New

Tutorials  --> Getbootstrap --> W3Schools

The BS navbar has two primary sections:

(1) the nav brand section and
(2) the nav link section

The default display of a BS navbar uses the <nav> tag with a few BS classes for default styling.
For example, <nav class="navbar navbar-expand-sm navbar-dark bg-dark">

Read through the examples in detail for how to (a) use the navbar in a container, (b) change when the nav links expand, (c) change the color of the navbar, (d) add an image to the brand section, (e) make the navbar fixed at the top, and lots and lots of other options.
